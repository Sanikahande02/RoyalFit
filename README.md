🏋️RoyalFit
---
A simple Gym Management System built using PHP & MySQL.
Features include:

* User Registration
* Login System
* BMI (Body Mass Index) Calculator
* Member Dashboard (basic)
* Admin-ready structure
  
---

# 🚀 Features

## 👤 User Module

* User Registration
* User Login Authentication
* User Dashboard
* Membership Details
* Logout Functionality

---

## 👨‍💼 Admin Module

* Admin Login
* Add New Members
* View Members
* Edit Member Details
* Delete Members
* Attendance Management
* Payment Management
* BMI Calculator
* Dashboard Statistics
